# Musicode
Create music by a simple code
## How to use
1. Create a virtual enviroment of python
2. Clone files into your virtual enviroment
3. Run `pip install -r requirements.txt`
4. Create Musicode file using your favorite text editor (We are planning to create a editor for Musicode)
5. Run `main.py` and select your Musicode file that you created
